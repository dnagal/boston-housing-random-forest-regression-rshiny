Ensuring reproducible results using set.seed(123) and splitting the bh_clean dataframe into train and test splits for machine learning

Random Forest Regression Model using the Train dataset. Also known as training the model.

The outcome variable is medv 500 trees will be used for this analysis Feature importance is included given the last line of code

rf_model <- randomForest(
  medv ~ .,
  data = train_clean,
  ntree = 500,
  mtry = floor(sqrt(ncol(train_clean)-1)),
  importance = TRUE
)
